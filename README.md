# このスクリプトについて
このスクリプトはGPUのファンを制御するためのものです

# 使い方
fan_temp_config
fan_speed_config
に数字を入れます

## 例

```
fan_temp_config=(40 45 50 55 60)
fan_speed_config=(30 35 40 45 100)
```
この場合は
| GPU温度 | ファンの回転数                                 | 
| ------- | ---------------------------------------------- | 
| ~40     | AUTO (nvidia-settings -a GPUFanControlState=0) | 
| 40~45   | 30                                             | 
| 45~50   | 35                                             | 
| 50~55   | 40                                             | 
| 55~60   | 45                                             | 
| 60~     | 100                                            | 

> [!TIP]
> 末尾の数字は~40を除いて以下になります
